# TPI – Gestión de Datos de Países

Trabajo Práctico Integrador de la materia **Programación 1 – 2025 (TUPAD)**.  
El proyecto consiste en una aplicación desarrollada en **Python 3**, que permite gestionar información sobre países, aplicando **listas, diccionarios, funciones, condicionales, filtros, ordenamientos y estadísticas** a partir de un archivo CSV.

---

## Objetivo del proyecto
Implementar un sistema que permita administrar un conjunto de países con sus datos básicos (nombre, población, superficie y continente), aplicando estructuras de datos y operaciones típicas de análisis.

---

## Instrucciones de uso

1. Descargar o clonar el repositorio completo.  
2. Asegurarse de que los archivos `tpi_gestion_paises.py` y `paises_base.csv` estén en la **misma carpeta**.  
3. Ejecutar el programa desde la terminal o consola con el comando:

   ```bash
   python tpi_gestion_paises.py
4. Seguir las instrucciones del menú principal para realizar búsquedas, filtros, ordenamientos, estadísticas o agregar/actualizar países.
5. Utilizar las opciones 7 o 9 para guardar los cambios en el archivo CSV.

## Funcionalidades principales
Cargar y guardar datos desde archivo CSV.

Agregar y actualizar países.

Buscar por nombre (parcial o exacto).

Filtrar por continente, rango de población o superficie.

Ordenar países por nombre, población o superficie (ascendente o descendente).

Mostrar estadísticas generales:

País con mayor y menor población.

Promedio de población y superficie.

Cantidad de países por continente.
